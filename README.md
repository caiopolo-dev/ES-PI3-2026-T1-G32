# MesclaInvest - Projeto Integrador 3

## Descrição do Projeto
O **MesclaInvest** é uma plataforma voltada à simulação de um ambiente digital de investimento. O objetivo do projeto é proporcionar uma reflexão prática sobre a tokenização de startups vinculadas ao ecossistema **Mescla** da PUC-Campinas.

Através do aplicativo móvel, os usuários podem visualizar startups, consultar, e realizar a compra e venda simulada de tokens.

> Todas as operações, saldos e tokens são **simulados**, não possuindo valor financeiro real ou integração com sistemas bancários.
